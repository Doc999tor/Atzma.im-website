# Usage
### Clone & Update
```
For clone with submodules add key '--recursive'
Update submodules 'git submodule update --recursive --remote'
```
### Development
```
npm install
npm start
Open http://localhost:3000
```
### Build
```
npm run build
```
### Build for production
```
npm run build-p
```
